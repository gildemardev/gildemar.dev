Next.js é um poderoso framework React que facilita a criação de aplicações web modernas. Neste post, vamos explorar algumas de suas principais características e como começar a usá-lo.

## O que é Next.js?

Next.js é um framework React que oferece:

- Renderização do lado do servidor (SSR)
- Geração de site estático (SSG)
- Roteamento baseado em sistema de arquivos
- Suporte a TypeScript
- Otimização de imagem automática

## Começando com Next.js

Para iniciar um novo projeto Next.js, você pode usar o seguinte comando:

```bash
npx create-next-app@latest meu-projeto-nextjs
```
